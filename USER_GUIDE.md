# 🖥️ GUIDE D'UTILISATION

## 🐧 Utilisation Linux  

* Connexion root

Script exécuté depuis le terminal.

Dans le même dossier que le script, lancer la commande suivante :

```bash
./administration.sh
```

## 🪟 Utilisation Windows

* Connexion Administrateur

Script exécuté depuis le terminal (PowerShell 7!)

Dans le même dossier que le script, lancer la commande suivante :
```PowerShell
.\administration.ps1
```
